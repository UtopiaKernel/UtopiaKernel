# 公益宣传：行动建议

技术工作者也可以用很小的成本，做一些长期有效的善意行动。以下是一些不带立场、可持续、可执行的建议（你可以把本页改成更符合项目调性的版本）。

## 1. 支持开放教育与开源

- 把你写过的教程/笔记整理成可公开访问的文档
- 给新手友好的 issue 回答与入门指引
- 维护一个“新手可做”的任务列表（good first issue）

## 2. 关注数字安全与隐私

- 推广强密码与密码管理器
- 帮助身边的人启用双因素认证（2FA）
- 提供可复用的安全配置模板（例如最小权限、备份策略）

## 3. 力所能及的捐助与志愿

- 定期小额捐助（比一次性大额更可持续）
- 参与本地志愿活动（支教、科普、社区服务等）
- 选择透明度更高、披露更充分的公益组织
