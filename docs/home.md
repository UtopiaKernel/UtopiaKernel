# UtopiaKernel 文档

欢迎来到 `UtopiaKernel` 的文档站模板。它是一个**纯静态、免构建**的项目文档框架，适合直接部署到 GitHub Pages 项目站点：

- ![UtopiaKernel 横幅](../assets/images/banner.svg)

- 目标站点：`https://utopiakernel.github.io/UtopiaKernel/`
- 内容来源：`docs/*.md`
- 导航配置：`docs/nav.json`

> 说明：本模板使用 CDN 引入 `marked`（Markdown 渲染）与 `highlight.js`（代码高亮）。如果你更偏好“完全离线/不依赖 CDN”，后续也可以把依赖改成本地文件。

## 你现在能做什么

- 在左侧导航中切换页面
- 在顶部搜索框中输入关键词（会在已配置的页面中做快速检索）
- 右上角按钮切换深浅色
